CommonsenseT2I: Text-to-Image Generation that Requires Commonsense Reasoning:
An Adversarial Challenge

►University of Pensylvania ►University of California, Santa Barbara
*Indicates Equal Contribution

Abstract

We present a novel task and benchmark for evaluating the ability of text--to-image(T2I) generation models to conduct visual-commonsense reasoning, which we call CommonsenseT2I.

  • Given two adversarial text prompts for image generation--both contain an identical set of action words with minor differences, we focus on evaluating whether image generation models can output images that fit commonsense knowledge.
    • E.g. prompt1: A bear eating salmon. v.s. prompt2: A person eating salmon.
  • We provide pairwise expected output descriptions for evaluation purpose, along with an automatic evaluation pipeline to check whether the the generated images match the expected commonsense reasoning result.
    • E.g. description1: Raw whole salmon fish. v.s. description2: Cooked salmon fillet.
  • We propose an automatic evaluation metric that is specifically designed for the task. Experiments validate that the proposed evaluation metrics are highly correlated with human perception.
  • We benchmark a range of state-of-the-art text-conditioned image generation models and find that, there is still a large gap between model performance and what humans can do.
The benchmark dataset is carefully hand-curated by expert annotators and is labeled with a rich set of fine-grained tags, such as category of commonsense and degree of objectivity, to assist in analyzing model performance. We aim for CommonsenseT2I to serve as a useful evaluation benchmark, fostering advancements in state-of-the-art technologies and driving further progress in the field.
